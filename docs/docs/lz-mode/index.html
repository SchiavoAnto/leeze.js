lz-mode

The attribute lz-mode is used to tell Leeze.js the mode of operation for that reactive element.
Supported values for this attribute are:

  • "text" (default)
  • "content"

If this attribute is not specified, Leeze.js will fall back to the default "text".

When updating the reactive element's content, Leeze.js checks this attribute to decide how to update the contents: when "content" is specified, Leeze.js will change the innerHTML property, allowing HTML to be set as content; when not used or "text" is specified, the library will change the textContent property of the reactive element.
