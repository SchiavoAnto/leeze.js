lz-value

The attribute lz-value is used to specify the name of the variable that Leeze.js associates with that element.
In other words, when the value of that variable changes, Leeze.js automatically updates all elements that used that variable name in their lz-value attribute.

<p lz-reactive lz-value="myVariable"></p>

This is an example of how to use lz-value. Here we tell Leeze.js to update the paragraph's content whenever the variable "myVariable" changes.

You can technically use both lz-value and lz-source on a reactive element but there is no guarantee that they both will work.

Note that variable in this case means a Leeze.js variable, not a standard JS one. Check LZ.setValue() for more information.
