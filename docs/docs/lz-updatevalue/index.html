LZ.updateValue()

The LZ.updateValue() function changes the specified variable by the specified delta. It is intended to be used with numbers.
The function has the following signature:

updateValue(name, delta)

Where:

  • name {string} is the name of the variable to change.
  • delta {number} is the desired change amount.

You can check out the counter example for more information on how to use this function.
