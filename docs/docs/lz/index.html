The LZ object

The LZ object contains all the functions necessary for Leeze.js to work. Some of these functions are public facing (meaning you can use them) and others are not.

You can use all the functions you see documented on this website.

Internal functions, which are not intended for public use, are not and will not be not documented here. You can still access them from the LZ object but support for them is not guaranteed and their behaviour is likely to change without notice.
