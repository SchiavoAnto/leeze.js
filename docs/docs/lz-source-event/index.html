lz-source-event

The attribute lz-source-event is used to specify to which event of the source element (specfied with the lz-source attribute) Leeze.js must hook to.
If you don't use this attribute, the oninput event will be used by default.

As opposed to lz-source, lz-source-event must be specified on the source element and NOT on the reactive element.
